java逻辑运算赋值快捷方式

2001-02-08 :: java

java中不存在&&=，取而代之的是&=。

1
2
3
 
boolean f = true, b = true;
  ⋮
f = f && b;

可以写成：

1
2
3
 
boolean f = true, b = true;
  ⋮
f &= b;

但后者诸位操作，不会进行短路。
