deploying pagila database

2003-08-13 :: postgres, postgresql, database, pagila

pagila is a popular postgres tutorial database.

  1. make sure superuser postgres existing, or:
# create role postgres superuser password '888';
  1. create database pagila:
# create database pagila with owner postgres encoding 'UTF8';
  1. create pagila schema:
$ psql -U postgres -f /xi/pagila-schema.sql pagila
  1. import data:
$ psql -U postgres -f /xi/pagila-data.sql pagila
